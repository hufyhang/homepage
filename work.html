Some recent works of mine.

What Map

A social network service powered by HTML5 Geolocation API, Offline Application Cache, and Leaflet Map. The whole Web app is built upon ChopJS framework, and uses ChopJS to access OpenStreetMap APIs to convert geolocation data into real address and vice versa.

Typie Chatie

An online chat room implemented by using WebSocket, and powered by a Node.JS backend.

Hyperportal

A Web portal and search engine for all the composite services published in HyperMash. It is powered by ChopJS framework, and supports URL deep linking.
